Space Station 13 - Modules - Types — Define Details

code/__DEFINES/clicks.dm

LEFT_CLICKThe left mouse button. Used mainly as params[BUTTON_CHANGED] == LEFT_CLICK, or params[LEFT_CLICK] = TRUE|FALSE if needed.
MIDDLE_CLICKThe middle mouse button. Used mainly as params[BUTTON_CHANGED] == MIDDLE_CLICK, or params[MIDDLE_CLICK] = TRUE|FALSE if needed.
RIGHT_CLICKThe right mouse button. Used mainly as params[BUTTON_CHANGED] == RIGHT_CLICK, or params[RIGHT_CLICK] = TRUE|FALSE if needed.
BUTTON_CHANGEDThe mouse button that triggered this Click() action. Used as params[BUTTON_CHANGED] = LEFT_CLICK|MIDDLE_CLICK|RIGHT_CLICK.
BUTTON_DRAGGEDThe mouse button that's being used for a drag action, only used in MouseUp(), MouseDown() Used as params[BUTTON_DRAGGED] = LEFT_CLICK|MIDDLE_CLICK|RIGHT_CLICK.
CTRL_CLICKTRUE if ctrl was held during this click action.
CTRL_CLICKEDUsed as params[CTRL_CLICKED] == TRUE|FALSE.
SHIFT_CLICKEDTRUE if shift was held during this click action. Used as params[CTRL_CLICKED] == TRUE|FALSE.
ALT_CLICKEDTRUE if alt was held during this click action. Used as params[CTRL_CLICKED] == TRUE|FALSE.
SCREEN_LOCThe pixel coodinates of the Click() in screen-loc format. ("[tile_x]:[pixel_x],[tile_y]:[pixel_y]")
ICON_XThe pixel coordinates of the Click() within the icon's coordinates. Ranges from 1 - icon.Width().
ICON_YThe pixel coordinates of the Click() within the icon's coordinates. Ranges from 1 - icon.Height().
VIS_XThe screen coordinates of the Click() relative to the icon's lower left corner. Will be empty if it's the same as ICON_X. Very finnicky, but here for completeness.
VIS_YThe screen coordinates of the Click() relative to the icon's lower left corner. Will be empty if it's the same as ICON_Y. Very finnicky, but here for completeness.
DRAG_CELLIf dragging a grid control, will return the name of the cell being dragged.
DROP_CELLIf dragging a grid control, will return the name of the cell being dropped on.
OVER_LINKIf the mouse is over a maptext link, OR if the mouse action is related to clicking a link. Used as params[OVER_LINK] == TRUE|FALSE
